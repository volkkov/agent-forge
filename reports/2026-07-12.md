## Flagged as hype

- **aura** — The project is heavily saturated with high-concept buzzwords like 'consciousness modules', 'phenomenal consciousness', and 'affective steering' that lack concrete, reproducible benchmarks. While it provides a specific 'evidence map' and a runnable demo, the core claims rely on complex theoretical frameworks (IIT 4.0, GWT) being applied to LLMs in a way that is largely speculative and unverified by standard AI benchmarks. The 'All Rights Reserved' license also makes it difficult to actually use or build upon.
- **swarm-ecosystem** — The README is almost entirely comprised of marketing buzzwords ('groundbreaking', 'lightspeed', '100% uptime', 'cutting-edge') and redirect links to other repositories rather than providing technical implementation details or documentation for this specific meta-repo. It functions more as a landing page for a corporate brand than a technical resource.
- **NewsAgent** — The README relies heavily on 'enterprise-grade' buzzwords and marketing fluff for enterprise support, while the actual technical implementation is a simple wrapper around NewsAPI and an LLM. It is not a duplicate, but the 'features' described are standard for any LLM/API integration.
